## 🐳 Docker Setup (Opcional)

Para ejecutar la aplicación completa con Docker:

```bash
# Inicio rápido
./docker-start.sh

# O con Makefile
make docker-up
```

Incluye:
- Next.js con hot reload
- PostgreSQL local (opcional)
- Adminer (UI de base de datos)
