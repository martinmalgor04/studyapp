## 3.1.1 - Variable `today` Duplicada

**Fecha**: 27 Enero 2026  
**Severidad**: 🔴 Critical  
**Módulo**: Dashboard

### Síntoma

Build error:

```
Error: the name `today` is defined multiple times
```

### Causa Raíz

En `dashboard.ts`, la variable `today` se declaraba dos veces:

```typescript
// Línea 43
const today = new Date();

// ... código ...

// Línea 54 - ❌ DUPLICADO
const today = new Date();
```

### Solución

```typescript
// Renombrar segunda instancia
const now = new Date();
```

### Lecciones Aprendidas

1. Usar nombres descriptivos evita colisiones
2. TypeScript strict mode detecta estos errores
3. Build errors son más estrictos que dev server

## 3.3.1 - Session Generation Logic Clarification

**Fecha**: 27 Enero 2026  
**Severidad**: 🟠 High  
**Módulo**: Session Generator (Logic)

### Síntoma

Confusión sobre cómo generar sesiones para diferentes tipos de exámenes:
- ¿Parciales vs Finales?
- ¿Desde cuándo contar los intervalos?
- ¿Cuántas sesiones por dificultad?

### Solución

Clarificación y documentación de algoritmos:

**Decisiones tomadas**:
1. **4 sesiones universales** (no varía por dificultad)
2. **Intervalos Anki**: [1, 3, 7, 14] días
3. **Dificultad afecta duración**, no cantidad
4. **Dos modos**: PARCIAL (desde clase) y FREE_STUDY (desde hoy)

### Archivos Creados

- Ver `AGENTS.md` para la lógica de generación de sesiones
- Actualización de `AGENTS.md` con algoritmos
